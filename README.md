# Mizuki Bot Help Documentation

这是 [Mizuki Bot 帮助文档](https://help.mizuki.top) 的源码仓库，基于 [VitePress](https://vitepress.dev/) 搭建。

## 文档包含什么

* **音游功能**：PJSK、舞萌 DX、中二节奏的查分查榜教程
* **账号绑定**：落雪咖啡屋绑定、成绩同步使用方法
* **其他功能**：表情包制作、经济系统、MC 服务器、B 站动态订阅
* **规范文件**：群规、免责声明、用户协议、赞助榜

## 技术栈

* **框架**：[VitePress](https://vitepress.dev/)（Vue 3）
* **环境**：Node.js v20+
* **部署**：Cloudflare Pages + GitHub Actions 自动部署


## 部署流程

推送到 `main` 分支会自动触发 GitHub Actions 构建并部署到 Cloudflare Pages。也可以在 Actions 面板手动触发。
