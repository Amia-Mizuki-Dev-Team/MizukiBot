# Mizuki Bot Help Documentation

Amia-Mizuki-Dev-Team/MizukiBot 是 [Mizuki Bot 帮助文档](https://help.mizuki.top) 的源码仓库。

它承载 Mizuki Bot 的用户文档、功能说明、使用规范、更新公告和关联项目介绍。站点基于 [VitePress](https://vitepress.dev/) 构建，通过 GitHub Actions 与 Cloudflare Pages 自动部署。

## 项目定位

本仓库不是 Mizuki Bot 的核心服务端，也不包含 Bot 运行时逻辑。它是 Mizuki Bot 生态的公开文档入口，用于让用户更快理解功能、完成绑定、查找规则，并跟踪相关项目状态。

文档站应保持三个原则：

- **清晰**：入口明确，页面标题和导航不制造额外理解成本。
- **克制**：减少模板化说明和重复文案，把重点留给实际使用场景。
- **可维护**：新增功能文档时同步维护导航、侧边栏与首页入口。

## 文档范围

- **入门指南**：基础使用、账号绑定、问题反馈与常见操作。
- **音游功能**：PJSK、舞萌 DX、中二节奏相关的查分、查榜和同步说明。
- **通用功能**：表情制作、经济系统、娱乐工具、B 站动态订阅等功能文档。
- **服务器与公告**：Minecraft 服务器说明、Bot 更新日志和站点公告。
- **规范文件**：群规、免责声明、用户协议、赞助信息与开发团队说明。
- **关联项目**：Mizuki Bot 生态内的插件、网关、同步工具和其他开源项目。

## 技术概览

- **文档框架**：VitePress
- **前端基础**：Vue 3
- **运行环境**：Node.js 20+
- **部署平台**：Cloudflare Pages
- **自动化流程**：GitHub Actions

## 维护方向

后续维护建议优先围绕站点的信息结构、搜索体验、移动端可读性与无障碍细节展开。视觉层面应减少过度装饰，保留 Mizuki Bot 的识别度，同时让内容本身成为页面重心。
