# 关联项目

这里收录 MizukiBot-Develop 组织内与 Mizuki Bot 生态相关的公开项目。页面用于帮助用户和开发者了解各项目的职责、使用场景和维护入口。

## 推荐查看

<div class="mzk-card">

### LXNS B50

舞萌 DX 查分与分析插件，集成落雪 (LXNS) 与水鱼 (Diving-Fish) 双数据源，覆盖 B50、AP50、单曲详情、战力分析、上分推荐和群内猜歌等场景。

[查看项目说明](/projects/lxns-b50)

</div>

<div class="mzk-card">

### Meme 表情插件

Mizuki Bot 的表情包管理插件，支持本地图库、图片上传、删除、溯源、文件夹锁定和多格式缓存管理。

[查看项目说明](/projects/mizuki-plugin-meme)

</div>

<div class="mzk-card">

### Auto Markdown

面向 Mizuki Bot 官方版的原生 Markdown 转换插件，用于把常见 URL 转换为 QQ 原生 Markdown 展示，并对 Mai-Sync 场景做适配。

[查看项目说明](/projects/mizuki-plugin-auto-markdown)

</div>
