# 关联项目

这里收录 Mizuki Bot 生态中与用户功能、官方 Bot 基础设施、Web 查询和移动端工具相关的公开项目。页面用于帮助用户和开发者了解各项目的职责、使用场景和维护入口。

## 推荐查看

<div class="mzk-card">

### LXNS B50

舞萌 DX 查分与分析插件，集成落雪 (LXNS) 与水鱼 (Diving-Fish) 双数据源，覆盖 B50、AP50、单曲详情、战力分析、上分推荐和群内猜歌等场景。

[查看项目说明](/projects/lxns-b50)

</div>

<div class="mzk-card">

### Mizuki Sync Android

面向舞萌 DX 玩家设计的 Android 第三方客户端，使用 LXNS API 与后端数据清洗服务提供成绩查询、曲库筛选和玩家状态同步。

[查看项目说明](/projects/mizuki-sync)

</div>

<div class="mzk-card">

### HX-Pjsk-Gateway

Project Sekai Web 查询网关，把 OneBot v11 数据节点转换为网页端可用的 API 服务。

[查看项目说明](/projects/hx-pjsk-gateway)

</div>

<div class="mzk-card">

### Mizuki Economy

群聊经济与社交互动插件，适配 QQ 官方机器人原生 Markdown 与内联按钮，同时保留非官方实例降级能力。

[查看项目说明](/projects/mizuki-economy)

</div>

<div class="mzk-card">

### Mizuki Plugin QBind

QQ 官方机器人环境下的真实 QQ 号绑定插件，用于建立会话虚拟 ID 与真实 QQ 号的映射。

[查看项目说明](/projects/mizuki-plugin-qbind)

</div>

<div class="mzk-card">

### Gensokyo NewQQ

Mizuki 官方 Bot 使用的 QQ 官方机器人客户端与协议桥接组件，承载 BotGo、HTTP API、WebSocket、WebUI 和 ID 映射等能力。

[查看项目说明](/projects/gensokyo-newqq)

</div>

<div class="mzk-card">

### Meme 表情插件

Mizuki Bot 的表情包管理插件，支持本地图库、图片上传、删除、溯源、文件夹锁定和多格式缓存管理。

[查看项目说明](/projects/mizuki-plugin-meme)

</div>

<div class="mzk-card">

### PJSK 车队广播

面向 PJSK 车队场景的自动广播、车牌识别和状态管理插件。

[查看项目说明](/projects/mizuki-plugin-pjskcarteam)

</div>

<div class="mzk-card">

### HAM 无线电日志

HAM 通联记录管理插件，支持呼号注册、QSO 日志、中继查询和导出。

[查看项目说明](/projects/mizuki-plugin-homo-qso)

</div>
