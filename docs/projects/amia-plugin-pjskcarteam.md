# Amia Plugin PJSK CarTeam

> PJSK 协力车队广播与房间状态辅助插件，当前为部分实现。

当前代码已实现房间号和备注解析、允许群管理、跨群广播、满员标记与过期清理；尚未实现成员加入/退出、成员列表和完整车队管理。

## 仓库

- GitHub: [Amia-Mizuki-Dev-Team/Amia-plugin-pjskcarteam](https://github.com/Amia-Mizuki-Dev-Team/Amia-plugin-pjskcarteam)

## 功能特性

- 识别 5 位房间号、`q1` 至 `q4` 缺人数和备注；
- 识别龙、虾、Sage、独轮、周回、填充等标签；
- 解析战力、小时、具体时间和“长途”备注；
- 在允许群之间广播当前房间列表；
- 使用 `m`、`满了` 或 `满` 标记房间已满；
- 定时清理过期或长时间未更新的房间；
- 群主/管理员控制本群广播，超级用户管理目标群。

## 使用说明

### 开/关广播

| 指令 | 说明 |
| :--- | :--- |
| `开启车队广播` | 允许本群接收/发送车队（群主/管理员） |
| `关闭车队广播` | 关闭本群车队广播（群主/管理员） |
| `添加车队群 [群号]` | 超管添加指定群 |
| `删除车队群 [群号]` | 超管移除指定群 |

### 发车

在已开启广播的群中发送格式：

```
[5位房间号] [描述] q[1-4] [备注]
```

示例：`12345 龙 q3 2小时 240`

| 参数 | 说明 |
| :--- | :--- |
| 房间号 | 5位数字（必填） |
| 描述 | 车型等信息 |
| `q[1-4]` | 缺几人（必填） |
| 备注 | 战力要求、时限等 |

### 满员

发送 `m` / `M` / `满了` / `满` 标记当前车队满员。
