# Amia Plugin PJSK CarTeam

> PJSK 协力车队自动广播插件，支持车牌识别、状态管理、跨群广播。

## 仓库

- GitHub: [Amia-Mizuki-Dev-Team/Amia-plugin-pjskcarteam](https://github.com/Amia-Mizuki-Dev-Team/Amia-plugin-pjskcarteam)

## 功能特性

- 自动识别 PJSK 协力房间号（5位数字）
- 多群实时广播车队信息
- 车型智能识别（龙/虾/Sage/独轮/周回/填充）
- 战力解析（支持综合力、百分比等多种格式）
- 时限设置（X小时/Y点）
- 满员自动标记与通知
- 定时清理过期车队
- 群管理员可自助开关广播

## 使用说明

### 开/关广播

| 指令 | 说明 |
| :--- | :--- |
| `开启车队广播` | 允许本群接收/发送车队（群主/管理员） |
| `关闭车队广播` | 关闭本群车队广播（群主/管理员） |
| `添加车队群 [群号]` | 超管添加指定群 |
| `删除车队群 [群号]` | 超管移除指定群 |

### 发车

在已开启广播的群中发送格式：

```
[5位房间号] [描述] q[1-4] [备注]
```

示例：`12345 龙 q3 2小时 240`

| 参数 | 说明 |
| :--- | :--- |
| 房间号 | 5位数字（必填） |
| 描述 | 车型等信息 |
| `q[1-4]` | 缺几人（必填） |
| 备注 | 战力要求、时限等 |

### 满员

发送 `m` / `M` / `满了` / `满` 标记当前车队满员。
