# Amia Plugin Meme

> Mizuki Bot 表情包管理系统插件，支持本地图库、自主上传、权限管理。

当前状态：仓库和基础说明已存在，插件主体仍待正式开发。下列能力以代码实际实现为准，不能将规划内容视为已上线功能。

## 仓库

- GitHub: [Amia-Mizuki-Dev-Team/Amia-plugin-meme](https://github.com/Amia-Mizuki-Dev-Team/Amia-plugin-meme)

## 功能特性

目标范围：随机、上传、删除、搜索、分类、来源记录、文件夹锁定、管理员权限检查、本地缓存和基础测试。

## 指令列表

### 基础指令

| 指令 | 说明 |
| :--- | :--- |
| `表情帮助` / `meme帮助` | 查看帮助菜单 |
| `发送 [文件夹]` | 随机发送一张指定分类的表情 |
| `随机meme` / `随一张` | 全局随机表情 |
| `查看 [图片名]` | 精确查看图片 |
| `看所有 [文件夹]` | 生成九宫格预览 |
| `表情列表` / `meme list` | 列出所有分类 |

### 上传系统

| 指令 | 说明 |
| :--- | :--- |
| `[文件夹] 上传` | 配图或回复 ZIP 存入 Bot 图库 |
| `[文件夹] 强制上传` | 强制上传（跳过重复检测） |

### 管理指令

| 指令 | 说明 |
| :--- | :--- |
| `最近上传` | 查看最新入库记录 |
| `删除 [图片名]` | 删除图片及记录 |
| `溯源 [图片名]` | 查看上传者信息 |
| `锁定/解锁 [文件夹]` | 控制文件夹权限 |
| `屏蔽群/解除屏蔽 [群号]` | 禁止某群使用 |
