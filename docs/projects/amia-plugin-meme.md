# Amia Plugin Meme

> Mizuki Bot 表情包管理系统插件，支持本地图库、自主上传、权限管理。

## 仓库

- GitHub: [Amia-Mizuki-Dev-Team/Amia-plugin-meme](https://github.com/Amia-Mizuki-Dev-Team/Amia-plugin-meme)

## 功能特性

- 支持 OneBot v11 / QQ 协议
- 自动缓存管理
- 多格式支持（jpg/png/gif/webp）
- 自动 WebP 转换
- 锁定/解锁文件夹权限
- 群黑名单屏蔽
- ZIP 批量导入
- 九宫格预览生成

## 指令列表

### 基础指令

| 指令 | 说明 |
| :--- | :--- |
| `表情帮助` / `meme帮助` | 查看帮助菜单 |
| `发送 [文件夹]` | 随机发送一张指定分类的表情 |
| `随机meme` / `随一张` | 全局随机表情 |
| `查看 [图片名]` | 精确查看图片 |
| `看所有 [文件夹]` | 生成九宫格预览 |
| `表情列表` / `meme list` | 列出所有分类 |

### 上传系统

| 指令 | 说明 |
| :--- | :--- |
| `[文件夹] 上传` | 配图或回复 ZIP 存入 Bot 图库 |
| `[文件夹] 强制上传` | 强制上传（跳过重复检测） |

### 管理指令

| 指令 | 说明 |
| :--- | :--- |
| `最近上传` | 查看最新入库记录 |
| `删除 [图片名]` | 删除图片及记录 |
| `溯源 [图片名]` | 查看上传者信息 |
| `锁定/解锁 [文件夹]` | 控制文件夹权限 |
| `屏蔽群/解除屏蔽 [群号]` | 禁止某群使用 |
