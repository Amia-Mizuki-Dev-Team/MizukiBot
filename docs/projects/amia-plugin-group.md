# Amia Plugin Group (公告)

> Mizuki Bot 智能公告系统插件，支持公告管理、多群自动广播与发送统计。

## 仓库

- GitHub: [Amia-Mizuki-Dev-Team/Amia-plugin-group](https://github.com/Amia-Mizuki-Dev-Team/Amia-plugin-group)

## 功能特性

- 公告内容管理（增/删/改/查）
- 群聊 + 私聊自动广播
- 防重复发送机制（内存锁 + 文件记录）
- 发送覆盖率统计
- 最多支持 5 条公告同时管理

## 指令列表

| 指令 | 说明 |
| :--- | :--- |
| `公告 查看` | 查看公告列表 |
| `公告 查看 [序号]` | 查看公告全文 |
| `公告 统计` | 查看最新公告的发送覆盖率 |
| `公告 增加 [内容]` | 新增公告（管理） |
| `公告 修改 [序号] [内容]` | 修改指定公告（管理） |
| `公告 删除 [序号]` | 删除指定公告（管理） |

公告新增或修改后，会在用户下一次互动时自动推送给对应群聊或私聊。
