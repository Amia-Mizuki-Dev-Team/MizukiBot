# LXNS B50

LXNS B50 是 MizukiBot-Develop 组织内的舞萌 DX 查分与分析插件。项目基于 NoneBot2，集成落雪 (LXNS) 与水鱼 (Diving-Fish) 双数据源，面向玩家成绩查询、战力分析、曲目检索和群内互动场景。

## 核心能力

<div class="mzk-card">

### 成绩查询

支持 `b50`、`ap50`、单曲详情、全服谱面统计和分数线计算。适合玩家快速查看 Best 50、AP 50、单曲达成率历史和目标达成容错。

</div>

<div class="mzk-card">

### 玩家分析

支持名片生成、Rating 曲线、双平台绑定状态诊断、查分数据源切换、战力排行、战力分析、近期成绩速报和高难底力分析。

</div>

<div class="mzk-card">

### 曲目检索与进度

支持查歌、谱面 ID 查询、定数查歌、BPM 查歌、版本完成表、等级进度和上分推荐。

</div>

<div class="mzk-card">

### 群内互动

支持猜歌、猜曲绘、听歌猜歌和别名猜歌等群内小游戏。

</div>

## 项目说明

官方机器人可使用 Markdown 高级按钮和快捷点击菜单；非官方机器人会自动降级为标准纯文本展示。项目还包含双源同步流程，会在每日凌晨自动同步歌曲数据并补齐曲绘资源。
