# Mizuki Plugin Maimai Expand

Mizuki Plugin Maimai Expand 是 MizukiBot-Develop 组织内的舞萌 DX 扩展插件项目，用于补充 Mizuki Bot 在舞萌 DX 查询、展示和周边能力上的实现。
